Hello, world!

git init ->inicia um repositorio local git

git status ->mostra a situação dos arquivos do seu repositório

git add ->adiciona um arquivo na area de staging

git commit -Adiciona os arquivos da área de stagins para o seu repositório local

git log ->mostra a listagem dos commits realizados

git ignore->Arquivos que devem ficar ocultos do git

git branch->Permite criar, listar, renomear e deletar branches

git checkout->Permite alternar entre branches criadas pelo git branch

git merge->Permite fazer o merge entre brenches

git clone->Cria uma cópia de um respositório git

git remote-->Especifica um repositório remoto para seu repositório local

git push -->Envia os commits do seu repositório local para o repostiório remoto

git pull-->Importa os commits do repositório remoto para seu repositório local e faz o merge no seu diretório de trabalho atual

git fetch-->Importa somente os índices do repositório remoto para seu repositório local
